## iSlide怎么安装？手把手教程

安装流程很简单，4步搞定。

1. 前往 iSlide 官方渠道下载安装包，选择对应系统版本（Windows 或 macOS）。
2. 运行安装程序，按提示完成安装。安装完成后，重新打开 PowerPoint 或 WPS，顶部菜单栏会自动出现「iSlide」选项卡。
3. 点击 iSlide 选项卡，选择「登录」，在弹出窗口中注册或登录账号。
4. 登录后点击头像进入「个人中心」，选择「兑换会员」，输入从数码荔枝购买的授权码，点击「确认兑换」即可激活。

支持的宿主：Windows 版 Microsoft Office 2010 至 365、WPS 个人版/专业版；macOS 版 Microsoft Office 2016 / 2019 / 365，以及 Keynote（部分功能）。
